📊 School Data Analysis & Performance Prediction
📌 Project Overview

This project analyzes school-level data to identify factors affecting student performance. It uses data analysis, visualization, and machine learning techniques to generate insights and predictions.
🎯 Objectives
Analyze student performance data
Identify key influencing factors
Build predictive models
Provide data-driven insights for education improvement
📂 Dataset

The dataset contains:

Academic scores (Math, Science, Language)
Infrastructure details (Electricity, Internet, Library)
Teacher-related features
Socio-economic indicators

🔧 Technologies Used
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Jupyter Notebook

📊 Key Steps
1. Data Cleaning
Handled missing values and ensured data consistency.
2. Exploratory Data Analysis (EDA)
Used charts like line plots, bar graphs, histograms, and heatmaps to understand patterns.
3. Feature Engineering
Encoded categorical variables and scaled numerical features.
4. Model Building
Applied models like Linear Regression and Random Forest.
5. Evaluation
Used R² Score and MSE for performance measurement.

📈 Key Insights
Teacher experience improves performance
Infrastructure impacts academic outcomes
Socio-economic factors influence learning


This project demonstrates how data analysis and machine learning can help improve educational outcomes.
